# Лабораторная 2 по дисциплине "Технологии программирования"

# Постановка задачи
Доработать веб-приложение интернет-магазина с системой покупок. 
	Условия:
		1 - Магазин товаров для автомобиля
		2 - При одновременной покупке двух экземпляров товаров разного вида покупателю должна предоставляться скидка 40% на любой третий товар.

# Краткое описание проекта
InternetShop - это небольшое веб-приложение для ,позволяющая купить товар с последующим сохранением в БД, построенное на Django. 
В InternetShop есть каталог товаров с возможностью добавления в корзину.
Товары добавляются в козину линейным способом.
Система скидок - автоматическое применение 40% скидки на третий товар при выполнении условия
Условие: если первые два товара разные, то скидка 40% применяется только к третьему товару.
Оформление заказов с сохранением данных клиента


# Средства реализации
1 - Python 3.8 - основной язык программирования
2 - Django 4.2.25 - веб-фреймворк
3 - PostgreSQL - система управления базами данных
4 - psycopg2 - драйвер для работы с PostgreSQL
5 - HTML5 - разметка страниц
6 - Django Templates - система шаблонов
